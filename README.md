## Installation

### Homebrew

```console
brew tap dennisbergevin/tools
brew install pwgo
```

### Go

Install with Go:

```console
go install github.com/dennisbergevin/pwgo@latest
```

Or grab a binary from [the latest release](https://github.com/dennisbergevin/pwgo/releases/latest).
